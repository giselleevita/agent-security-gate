# Company Brief

This repository converts the benchmark asset into a focused product wedge:

- Product: `Agent Security Gate`
- Primary problem: unsafe agent actions in CI and runtime
- Core value: pre-execution enforcement with measurable security outcomes
- MVP pillars:
  - policy gateway
  - security eval gate
  - approval and audit plane

## Intended buyer

- security engineering leaders
- platform teams shipping tool-using agents
- regulated teams needing evidence and human oversight

## Why this repo exists

The original benchmark repo proved the concept. This repo is the product scaffold that separates:
- benchmark logic from product messaging
- runtime enforcement from thesis framing
- approval and evidence workflows from purely academic evaluation
